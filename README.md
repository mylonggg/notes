# CloudNotes | 2026 隐私加密笔记与导航
## 创建后端
1.创建 Workers

2.创建 D1 SQL数据库 ，名字随意-[sql数据库](./src/sql.md)

3.编辑代码 [worker输入代码](./worker.md)

4.保存部署

5.绑定D1数据库，变量名称`DB`，数据库选择自己新建

6.绑定AI,变量名AI

7.绑定自定域，然后添加一个环境变量（密码用于绑定数据库）类型：密钥，变量名`ADMIN_KEY`,值设置成你想要的密码 8.再加一个变量`ACCESS_PASSWORD`，作为前端密码

## 创建前端
1.新建Pages 连接github账号，选项目名称，，直接部署，然后绑定自定域

2.初化设置，绑定后端的自定域，密码为：`ADMIN_KEY`的密码
